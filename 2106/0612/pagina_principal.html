    Um sistema operacional é um programa cujo objetivo é gerenciar os recursos do sistema, definir um sistema de arquivo, alocar a memória, gerenciar os processos do sistema, lidar com a entrada e saída de dados, etc., fornece também uma interface entre o computador e o usuário e controla como periféricos interagem com o hardware. Atualmente nos sistemas modernos os sistemas operacionais são iniciados por meio de outro programa, a BIOS, no entanto não é algo necessário.

    Existem vários sistemas operacionais, entre eles os mais famosos são: Windows, Linux e MacOS. Existem várias diferenças entre eles, no entanto ao meu ver uma das mais relevantes é sua licença de uso, enquanto o Windows e o MacOS são softwares proprietários, ou seja, empresas controlam a sua distribuição, no caso deles o Windows é controlado pela Microsoft e o MacOS é controlado pela Apple, já o Linux é um sistema operacional open source, ou seja, seu código fonte é disponível a todos, visto isso no Linux existem várias versões concorrentes do sistema, entre elas algumas famosas são: Ubuntu, Mint e Debian.
